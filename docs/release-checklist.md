# 절약지도 검수/배포 체크리스트

앱인토스 콘솔 제출 전에 복사해 쓸 문구와 확인 항목입니다.

## 앱 기본 정보

- 앱 이름: 절약지도
- 한 줄 설명: 내 주변 착한가격업소와 생활비 절약 정보를 빠르게 찾는 미니앱
- 카테고리 후보: 생활/편의, 혜택/생활
- 주요 사용자: 외식비와 생활 서비스 비용을 아끼고 싶은 토스 사용자

## 앱 설명 문구

절약지도는 공공데이터를 기반으로 주변의 착한가격업소와 생활비 절약처를 찾아주는 앱인토스 미니앱입니다.
사용자는 카테고리, 검색어, 저장 기능으로 필요한 가게를 빠르게 찾고, 상세 화면에서 주소와 전화번호,
지도 검색 링크를 확인할 수 있습니다. 방문 기록, 가게 메모, 최근 검색어를 기기에 저장해 자주 가는
절약처를 더 편리하게 관리할 수 있습니다. 금융 거래 내역, 결제 내역, 계좌 정보는 수집하지 않습니다.

## 핵심 기능

1. 지역 기반 절약처 목록
   - 공공데이터로 변환한 착한가격업소 목록을 보여줍니다.
   - 위치 권한을 허용하면 좌표가 보강된 업소를 가까운 순서로 보여줄 수 있습니다.
   - 홈 화면에서 내 위치 기준 가장 가까운 절약처를 즉시 확인할 수 있습니다.

2. 카테고리/검색 필터
   - 식비, 장보기, 교통, 생활 카테고리와 검색어로 결과를 좁힙니다.
   - 최근 검색어를 저장해 다시 입력 없이 재검색할 수 있습니다.

3. 저장 및 메모
   - 마음에 드는 절약처를 저장하고 다시 확인합니다.
   - 저장한 가게에 개인 메모(영업시간, 주차 여부 등)를 남길 수 있습니다.
   - 저장 목록을 텍스트로 공유하거나 클립보드에 복사할 수 있습니다.

4. 방문 기록
   - 다녀온 가게에 방문 체크를 남기고 마이 탭에서 방문 현황을 확인합니다.

5. 마이 탭
   - 저장한 가게의 카테고리 분포, 온누리 가맹 수, 방문 현황 등 통계를 제공합니다.
   - 데이터 신선도를 표시하고, 오래된 경우 안내 메시지를 보여줍니다.

## 앱 내 기능 후보

앱인토스 콘솔의 앱 내 기능 등록이 필요할 때 아래 후보를 사용합니다.

- 내 주변 착한가격업소 찾기
- 만원 이하 식사 찾기
- 저장한 절약처 보기

## 권한 사용 사유

위치 권한:

사용자 주변의 절약처를 가까운 순서로 보여주기 위해 현재 위치를 사용합니다.
위치 정보는 서버로 전송하거나 저장하지 않습니다.

## 외부 이동 안내

- 길찾기/지도 버튼은 네이버지도 또는 카카오맵 앱/검색 페이지로 이동합니다.
- 전화번호가 있는 업소는 기기 전화 앱으로 연결될 수 있습니다.

## 데이터 출처

- 행정안전부\_착한가격업소 현황: https://www.data.go.kr/data/3045247/fileData.do
- 소상공인시장진흥공단\_전국 온누리상품권 가맹점 현황: https://www.data.go.kr/data/3060079/fileData.do

현재 앱에 포함된 기본 데이터는 `전국` 기준이며, 최신 빌드 기준으로 약 13,000건 규모입니다.
착한가격업소는 주소 기반으로 좌표를 보강한 데이터가 대부분 반영되어 있고, 일부 미보강 주소는 목록 중심으로 계속 노출됩니다.
온누리상품권 CSV는 현행 원본 구조상 상세 주소 밀도가 낮은 항목이 있어, 지도 탐색보다 목록 검색/필터 용도가 더 적합합니다.

## 개인정보/저장 데이터

- 로그인: 사용하지 않음
- 서버: 사용하지 않음
- 금융 데이터: 수집하지 않음
- 위치 정보: 가까운 절약처 계산에만 사용, 서버 전송 및 저장 없음
- 저장 데이터: 아래 항목을 앱인토스 Storage 또는 `localStorage`에만 저장
  - `saving-map.saved-store-ids` — 즐겨찾기한 업소 ID 목록
  - `saving-map.visited-store-ids` — 방문 체크한 업소 ID 목록
  - `saving-map.recent-store-ids` — 최근 본 업소 ID 목록 (최대 5개)
  - `saving-map.search-history` — 최근 검색어 목록 (최대 8개)
  - `saving-map.store-notes` — 저장 업소별 개인 메모 (최대 80자)

## 출시 전 필수 확인

- `granite.config.ts`의 `appName`이 앱인토스 콘솔 앱 이름과 일치하는지 확인
- `granite.config.ts`의 `brand.icon` URL 접근 가능 여부 확인
  - 현재 설정: `public/icon.png` (600×600 정사각형 PNG) — GitHub raw URL로 연결
  - GitHub 저장소가 public이고 해당 파일이 push되어 있어야 URL이 유효함
  - `appsintoss-logo.png`(가로형 배너)는 아이콘으로 사용하지 않음
- 고객센터 이메일 또는 URL 준비
- 개인정보처리방침 URL 준비 (저장 데이터 항목 5가지 반영)
- `pnpm data:download` 후 원본 데이터가 최신인지 확인
- `pnpm data:build` 후 `public/data/stores.json`이 정상 생성되는지 확인
- `pnpm lint`, `pnpm build` 통과 확인

## 테스트 체크리스트

### 기본 동작

- 첫 실행 시 홈 화면이 뜨는지 확인
- `public/data/stores.json`이 200으로 로드되는지 확인
- 위치 권한 허용 시 앱이 멈추지 않는지 확인
- 첫 진입 시 위치 권한을 강제로 요청하지 않는지 확인
- 위치 권한 거부 시 지역 기준 목록으로 계속 사용할 수 있는지 확인
- 검색어 입력 후 결과가 줄어드는지 확인
- 카테고리 탭 선택 시 결과가 바뀌는지 확인
- 업소 상세 시트가 열리고 닫히는지 확인
- 즐겨찾기 저장 후 저장 탭에서 유지되는지 확인
- 앱 재실행 후 즐겨찾기가 유지되는지 확인
- 지도 검색 버튼이 외부 페이지를 여는지 확인
- 전화번호가 있는 업소에서 전화 연결 버튼이 동작하는지 확인
- 하단 탭 홈/지도/저장/마이 이동이 자연스러운지 확인

### 신규 기능

- 위치 허용 후 홈 탭에 가장 가까운 절약처 추천 카드가 노출되는지 확인
- 가게 검색 후 상세를 열었다가 닫으면 최근 검색어가 저장되는지 확인
- 최근 검색어 칩 탭 시 검색어가 채워지는지 확인
- 최근 검색어 개별 삭제 및 전체 삭제가 동작하는지 확인
- 상세 시트에서 방문 체크 토글이 동작하는지 확인
- 방문 체크 후 목록 카드에 '다녀옴' 배지가 표시되는지 확인
- 마이 탭에 방문 수와 저장 분석이 노출되는지 확인
- 저장된 가게 상세에서 메모 입력 후 앱 재실행 시 유지되는지 확인
- 저장 탭에서 공유 버튼이 동작하는지 확인 (share 또는 클립보드)
- 지도 탭에서 현위치 버튼 탭 시 내 위치로 이동하는지 확인
- 지도 목록 패널의 카드에 방문 배지와 메모가 표시되는지 확인
- 마이 탭 데이터 신선도 배지가 표시되는지 확인

### 빌드

- `pnpm build` 후 생성된 `.ait` 패키지로 테스트 업로드가 가능한지 확인

## 현재 남은 제품 리스크

- 일부 미보강 주소는 지도 핀과 거리 계산이 제한됩니다.
- 전국 데이터 전체를 한 번에 싣고 있어, 사용자 수가 늘면 지역 단위 분할 JSON 또는 서버 API 전환이 필요합니다.
- 온누리 데이터는 현행 원본 구조상 상세 위치 탐색 기능으로 쓰기 어렵습니다.
- 아이콘, 고객센터, 개인정보처리방침은 검수 전에 실제 값으로 교체해야 합니다.
